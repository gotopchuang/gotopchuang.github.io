About #




莊向峰，1994年生於台灣，主修計算機科學及資訊網路與多媒體,身兼藝術家及AI研究員的身份。
曾獲得2018台北國際攝影節新銳獎評審團特別獎。

以攝影為主，跨足錄像、裝置等媒材，創作主要討論數位化時代影像的「傳播機制」、「載體」
對影像觀念的轉化及對人類之間的互動關係與影響，

從技術運作原理出發，與哲學及社會做出連結。
